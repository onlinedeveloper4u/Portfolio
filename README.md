# Portfolio

A personal portfolio website built with React, TypeScript, and Supabase.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

You can deploy this project to any static hosting service like Vercel, Netlify, or GitHub Pages.
